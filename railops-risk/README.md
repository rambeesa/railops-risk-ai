# RailOps Risk

Rail safety risk prediction and operations tooling.

## Structure

- **backend/** — API and risk prediction logic
- **frontend/** — Web UI
- **data/** — Datasets and input data
- **models/** — Trained/saved models
- **docker-compose.yml** — Run backend and frontend with Docker

## Quick start

```bash
docker compose up --build
```
